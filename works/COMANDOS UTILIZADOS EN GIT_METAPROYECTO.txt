COMANDOS UTILIZADOS EN GIT_METAPROYECTOS



1- Descargar el metaproyecto: git submodule add --recursive --branch
2- Descargar el submodulo:  git submodule add
3- Descargar y combinar el proyecto git remoto: git pull
4- mover a los submodulos a una rama: git submodule foreach git checkout
5- Saber la rama actual: git branch
6- Actualizar todos los submodulos: git submodule update --init --recursive
